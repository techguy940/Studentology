Elementary Formulas

Area



Square:                l * 4
Rectangle:          2 * (l+b)
Triangle:              (h*b)/2
Circle:                   π * r²
Parallelogram:   a * b * sin(x)
Trapezoid:           ((a+b)/2) * h
Rhombus:            1/2 * d1 * d2

Volume



Cube:                a³
Cuboid:             l * b * h
Cylinder:           π * r² * h
Pyramid:           1/3 * b * h
Cone:                 1/3 * π * r² * h
Sphere:             4/3 * π * r³
Ellipsoid:           4/3 * r1 * r2 * r3

Squares



(a+b)² :           a² + 2ab + b²
(a-b)² :            a² - 2ab + b²
 a²-b² :             (a+b) * (a-b)
(a+b+c)² :     a² + b² + c² +
                                      2 * (ab + bc + ca)
(a-b-c)² :      a² + b² + c² -
                                      2 * (ab + bc - ac)

Cubes



(a+b)³ :          a³ + 3a²b + 3ab² + b³
(a-b)³ :          a³ - 3a²b + 3ab² - b³
 a³+b³ :           (a-b) * (a² - ab + b²)
 a³-b³ :           (a-b) * (a² + ab + b²)
(a+b+c)³ :   (a³+b³+c³) + 3 *
                                      [(a+b+c) *
                                   (ab+ac+bc) −
                                    abc]
